# VirusTotalTGBot
## Запуск
### Переменные окружения
#### Linux
``` bash
export TG_BOT_API="<API телеграм бота>"
export TG_BOT_NAME="<Имя телеграм бота (@имя)>"
export VT_API="<API Virus Total>"
```
#### Windows
``` bash
set TG_BOT_API=<API телеграм бота>
set TG_BOT_NAME=<Имя телеграм бота (@имя)>
set VT_API=<API Virus Total>
```
### Команда для запуска
`java -cp "bin/VTTGB.jar:lib/*" com.MyDo.Program`
